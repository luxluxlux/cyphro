# Changelog

All notable changes to **Cyphro** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and the project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] — 2026-03-01

### Added
- Client-side file protection with password-based access.
- Ability to disguise protected files as other file types.
- Fully browser-based processing with no server-side file handling.
- Support for common file formats (documents, images, audio, video).
- Image content moderation before file processing.
- Protection against processing high-risk executable file types.
- Clear user warnings for unsupported or restricted files.
- Open-source project structure with MIT license.
